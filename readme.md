# OpenLayers + Vite Kartenprojekt

Dieses Projekt zeigt die Verwendung des `ol` (OpenLayers) Pakets in Kombination mit [Vite](https://vitejs.dev/).

## Erste Schritte

Um das Projekt zu starten, benötigen Sie Node.js 14 oder höher. Führen Sie folgende Befehle aus:

```bash
git clone [repository-url]
cd my-app
npm install
```

## Entwicklung

Starten Sie den Entwicklungsserver (erreichbar unter http://localhost:5173):

```bash
npm start
```

## Produktions-Build

Erstellen Sie eine produktionsreife Version:

```bash
npm run build
```

Die Build-Dateien werden im `dist`-Verzeichnis erstellt. Sie können diese Dateien auf Ihren Server hochladen.

Zum Testen des Builds lokal:

```bash
npm run serve
```

## Features

- Interaktive OpenStreetMap Integration
- Responsive Design
- Zoom- und Pan-Funktionalität
- Layer-Management
